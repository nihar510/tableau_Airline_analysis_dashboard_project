# Tableau_airline_analysis_dashboard_project
Dashboard with airline analysis stats

The dashboard can be viewed at the following link: https://public.tableau.com/views/AirlineAnalysisDashboard_17735005965080/AIRLINEANALYSISDASHBOARD?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

This is a comprehensive airline (British Airways) review dashboard inspired by Mo Chen's British Airways analysis featuring parameter controls ('Pick a Metric') and interactive filters for traveller type, seat class, aircraft, and continent. Track satisfaction trends over time, identify top-performing aircraft (Boeing 747-4 at 4.7 rating), and pinpoint low-scoring service areas like entertainment (1.4 average) requiring attention 

Inspiration: https://public.tableau.com/views/BritishAirwaysReviews_17071660082360/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link
